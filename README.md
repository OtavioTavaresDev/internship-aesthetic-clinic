## 📖 Overview

During my internship at a Brazilian aesthetic clinic, I identified repetitive operational tasks and developed several internal tools to improve productivity, workflow organization, and customer follow‑up. These applications were built independently as learning projects and portfolio pieces.

**No confidential information, customer data, or proprietary company assets are included in this repository.**

---

## 🗂️ Projects

| Project | Description | Status |
|---------|-------------|--------|
| **Atlas AI** | Internal AI assistant for operational questions and customer support | MVP |
| **FlowCRM** | Simple CRM focused on clinic workflow and customer management | MVP |
| **FlowTasks** | Task management platform for daily workflow organization | MVP |
| **FollowFlow** | Automated follow‑up manager based on treatments performed | MVP |
| **QuickCounter** | Simple counter for repetitive operational tasks | MVP |

---

## 🔍 Project Details

### 🤖 Atlas AI
Internal AI assistant designed to help employees with operational questions and customer support.
- Knowledge base with clinic procedures
- Prompt engineering for accurate responses
- Internal documentation access
- Customer assistance capabilities

### 📊 FlowCRM
Simple CRM focused on clinic workflow.
- Customer management
- Appointment history tracking
- Analytics and reports

### 📋 FlowTasks
Task management platform.
- Daily task organization
- Priority settings
- Progress tracking
- Workflow optimization

### 🔄 FollowFlow
Automated follow‑up manager based on treatments performed.
- Follow‑up scheduling
- Message generation
- Task creation
- Daily reports

### 🔢 QuickCounter
Simple counter application created for repetitive operational tasks.
- Quick and intuitive interface
- Ideal for counting procedures or inventory
