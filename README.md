# Internship – Aesthetic Clinic Automation

During my internship at a Brazilian aesthetic clinic, I identified repetitive operational tasks and developed several internal tools to improve productivity, workflow organization and customer follow-up.

These applications were developed independently as learning projects and portfolio pieces.

No confidential information, customer data or proprietary company assets are included in this repository.

---

## Projects

### AI Assistant

Internal AI assistant designed to help employees with operational questions and customer support.

Features

- Knowledge base
- Prompt engineering
- Internal documentation
- Customer assistance

---

### FlowCRM

Simple CRM focused on clinic workflow.

Features

- Customer management
- Appointment history
- Analytics
- Reports

---

### FollowUp Manager

Automatic follow-up manager based on treatments performed.

Features

- Follow-up scheduling
- Message generation
- Task creation
- Daily reports

---

### FlowTasks

Task management platform.

Features

- Daily tasks
- Priorities
- Progress tracking
- Workflow organization

---

### QuickCounter

Simple counter application created for repetitive operational tasks.
